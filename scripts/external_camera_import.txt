
# connect your phone and pull geotracking data
`./geo_tag/pull.sh`


# convert raws to webp and geotag
in a directory of CR2 files:
```
~/repos/koo5/hillview/0/hillview/scripts/raw/raw.sh
```



# if desired, export any photos with settings different from defaults:
```
# export pictures manually in darktable
darktable .;

# overwrite the auto-developed files
cp darktable_exported/* webp/

# transfer tags again
~/repos/koo5/hillview/0/hillview/scripts/exif_tags_from_cr2_to_webp.sh

# geotag again
set correction -19
~/repos/koo5/hillview/0/hillview/scripts/geo_tag.sh ~/GeoTrackingDumps --correction $correction --parallel 40 webp/*

```



# upload
use `--version` when re-uploading.
```
~/repos/koo5/hillview/0/hillview/backend/debug.sh upload-files --parallel 40 (rp webp/*)
~/repos/koo5/hillview/0/hillview/backend/debug.sh upload-files --parallel 40 --skip-anonymization --version  (date --utc "+%s") (rp webp_noanon/*)
```
